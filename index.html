Rólam

Nagy Istvánnak hívnak, jelenleg egy egészségügyi intézménynél dolgozom, mint informatikus/rendszergazda, amikor éppen nem dolgozom akkor a programozás karrierem építésén tevékenykedem. Régi tervem, hogy kipróbálom magam az informatika egy másik területén, a programozásban is.

Tapasztalat

Néhány „dolog” amivel a munkám során találkozom:

  • Windows Server (AD, GPO, DNS, Fileserver, Webserver, FTP)
  • Hyper-V, VMware
  • CentOS (Postfix, Dovecot, Squirrelmail, Roundcube, Amavisd, SpamAssassin, Apache)
  • Microsoft SQL, MySQL, phpMyAdmin
  • Mikrotik (RouterOS, SwitchOS, Dude)
  • VPN (L2TP/IPSEC, SSTP, Wireguard, OpenVPN)
  • Synology
  • Lotus Domino/Notes
  • Joomla, Gantry5, Photoshop
